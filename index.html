PHASE 2 — Local AI Script

pip3 install requests import requests, os r = requests.post( "https://openrouter.ai/api/v1/chat/completions", headers={"Authorization": f"Bearer {os.getenv('OPENROUTER_API_KEY')}"}, json={"model":"openrouter/free","messages":[{"role":"user","content":"Explain Python"}]} ) print(r.json()["choices"][0]["message"]["content"])

PHASE 3 — Telegram AI Interface

Create bot via @BotFather

pip install python-telegram-bot # bot.py from telegram.ext import Updater, CommandHandler import requests, os def ask(update, context): q = " ".join(context.args) r = requests.post( "https://openrouter.ai/api/v1/chat/completions", headers={"Authorization": f"Bearer {os.getenv('OPENROUTER_API_KEY')}"}, json={"model":"openrouter/free","messages":[{"role":"user","content":q}] }) update.message.reply_text(r.json()["choices"][0]["message"]["content"]) updater = Updater("YOUR_TOKEN") updater.dispatcher.add_handler(CommandHandler("ask", ask)) updater.start_polling() updater.idle()
